# iOS development tips
Misc tips for iOS development

## Free HD space used by Xcode's derived data.
- Open terminal
- cd ~/Library/Developer/Xcode
- rm -rf DerivedData
- [Source](https://twitter.com/johnsundell/status/982274922528563200)
